## 📋 Цель исследования

Разработка и исследование комплексного подхода к детекции дефектов на поверхности стального листа (датасет Severstal), включающего:

1. **Всесторонний анализ неоднородности** исходных данных (текстуры, цвет, яркость, спектральные характеристики)
2. **Генерацию синтетических изображений** на основе Stable Diffusion img2img с Poisson-смешиванием и спектральным контролем
3. **Ablation study** для поиска оптимальных гиперпараметров генерации
4. **Сравнительный эксперимент** влияния синтетических данных на качество детекции (5 конфигураций датасетов × 3 стратегии обучения)
5. **Дистилляцию знаний** из тяжёлой модели LT-DETR + DINOv2 в лёгкие архитектуры (YOLOv8 Nano, PicoDet-S, Faster R-CNN R18) для промышленного внедрения
